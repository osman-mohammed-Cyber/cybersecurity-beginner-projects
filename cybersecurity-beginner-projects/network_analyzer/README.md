# Network Analyzer

## Overview
A beginner Python tool to check if a host (IP or domain) is reachable via ping.

## What This Project Does
- Resolves hostname to IP
- Sends a single ICMP ping request
- Shows if target is reachable

## Concepts Learned
- DNS resolution
- Ping (ICMP) basics
- Error handling
- Functions and conditionals

## Educational Purpose
Created to understand basic network reachability in a safe and educational way.

## How to Run
1. Install Python 3
2. Run: `python network_analyzer.py`
3. Enter hostname or IP

## Example Output
Enter hostname or IP: google.com
Target: google.com
IP: 142.250.xxx.xxx
Status: Reachable
