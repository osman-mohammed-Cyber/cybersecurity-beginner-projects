# Beginner Cybersecurity Projects Guide

## Overview
This guide summarizes three beginner-level Python projects designed to teach fundamental cybersecurity and networking concepts.  
The projects focus on practical learning in a safe and educational way.

---

## 1. Python Port Scanner

### What It Does
- Checks one or multiple ports on a host
- Uses Python sockets to detect open ports
- Displays results clearly

### Concepts Learned
- TCP/IP basics
- Socket programming
- Loops and conditionals

### Educational Purpose
Created for learning and understanding basic network port scanning safely.

### How to Run
1. Install Python 3
2. Run: `python port_scanner.py`
3. Enter host and port(s)

### Example Output
Target: 192.168.1.1
Port 22: Open
Port 80: Closed

---

## 2. Password Strength Checker

### What It Does
- Evaluates password length and character types
- Gives simple suggestions to improve strength

### Concepts Learned
- String handling
- Loops and conditionals
- User input validation

### Educational Purpose
Created to learn password security principles and provide educational feedback.

### How to Run
1. Install Python 3
2. Run: `python password_checker.py`
3. Enter a password

### Example Output
Enter password: MyPass123!
Strength: Medium
Suggestions: Add special characters

---

## 3. Network Analyzer

### What It Does
- Resolves a hostname to its IP address
- Sends a single ping (ICMP) request
- Indicates whether the target is reachable

### Concepts Learned
- DNS resolution
- Ping basics (ICMP)
- Error handling
- Functions and conditionals

### Educational Purpose
Created to understand basic network reachability in a safe and educational way.

### How to Run
1. Install Python 3
2. Run: `python network_analyzer.py`
3. Enter a hostname or IP

### Example Output
Enter hostname or IP: google.com
Target: google.com
IP: 142.250.xxx.xxx
Status: Reachable

---

## Summary
These projects are **educational tools** that provide hands-on experience in:

- Network communication basics
- Password security awareness
- Python programming for cybersecurity

They are designed for beginners to explore cybersecurity concepts safely and effectively.
