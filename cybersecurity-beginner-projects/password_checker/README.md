# Password Strength Checker

## Overview
A beginner Python tool to evaluate password strength and provide feedback.

## What This Project Does
- Checks password length and character types
- Gives simple suggestions to improve strength

## Concepts Learned
- String handling
- Loops and conditionals
- User input validation

## Educational Purpose
Created to learn password security principles and provide educational feedback.

## How to Run
1. Install Python 3
2. Run: `python password_checker.py`
3. Enter a password

## Example Output
Enter password: MyPass123!
Strength: Medium
Suggestions: Add at least one special character
